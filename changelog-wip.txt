BotC v1.5.2 - Feb XX, 2026
Changes and features:
- #45: Add an in-game link to the project's Issues page for easy bug reporting.
- #46 (partial): Add an in-game explainer on how to use the pack
- Added a screen to view all Travellers, which can be accessed from the script menu.
- Added a sound for when a regular vote is taken. I don't love this sound; it may change or be removed in the future, but for the time being it's helpful to keep track of vote counts.
- Added a note on the pause screen warning that the pack is in an early stage of development.
- Instead of big ugly banners, houses will now display the name and face of the player they belong to next to the entrance.
- Ghost vote sound can now be heard more clearly from a distance.
- Houses now have carpets and beds that closely match their assigned color. (thanks @boofsly!)
- Removed /traveler prompt on spectator join until that command actually exists.

- #52: Wiki Links in the Script do not work for characters who have a space in their name. (thanks @JakeHumphries4!)
- Changelog screen has no background.
- Greenhouse and mint house voice chats have slightly incorrect bounds.
- Items on the inn counter can be interacted with.
- Sunflowers point the wrong way.
- Grimoire "Actions Menu" button is positioned incorrectly at some resolutions.
